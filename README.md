# Internal HR support chatbot
A production-ready Retrieval-Augmented Generation (RAG) chatbot built with Streamlit, FAISS, and Mistral AI to chat with your local `.docx` files securely.

## Getting Started

Follow these steps to clone the repository and set up your local development environment.

### 1. Clone the Repository

Open your terminal or bash prompt and execute the following commands to clone the codebase and navigate into the project directory:

```bash
git clone [https://github.com/yourusername/internal_hr_chatbot.git](https://github.com/yourusername/internal_hr_chatbot.git)
cd internal_hr_chatbot
pip install -r requirements.txt
